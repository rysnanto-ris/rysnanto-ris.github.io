Code Editor Untuk Hp Dan Laptop

2021-03-20 — Penulis Rysnanto

Sebelum nya di artikel saya , kita kan membasah front-end webdev , nah kan untuk menjadi semorang developer bagian ini harus menguasai tiga bahasa pemrogaman yaitu html ,css dan javascript

Nah untuk alat ngoding nya sendiri gimana? Disini saya sudah merangkum alat ngoding yang ada di android dan di pc / laptop

Saya sebagai user android , saya dahulukan terlebih dahulu alat ngoding yang ada di android terbaik menurut saya ya ! Bukan menurut orang lain

Alat ini tentunya sudah menemani saya dulu sampai sekarang , yah sudah ku anggap sebagai keluarga saya :V

Yang pertama yaitu aplikasi yang bernama

Anwriter , nah jujur saja , pertama kali pakai apk ini tangan saya langsung jatuh cinta hehe , di coba sendiri ya , untuk fitur nya auto save , yang pasti ukuran nya juga ringan cocok buat hp kentang seperti hp saya :( , untuk fitur lainnya di baca sendiri di playstore ya om https://play.google.com/store/apps/details?id=com.ansm.anwriter DroidEdit , tampilan nya keren banyak fitur nya , fitur nya juga sama auto save ringan dan lain lain cek sendiri saja di playstore https://play.google.com/store/apps/details?id=com.aor.droidedit

Itu dia rekomendasi saya code editor paling enak dan ringan di android , sekarang kita menuju ke laptop / pc

Sublime Text , nah saya rekomendasikan pakai software ini aja , pas saya ada di sekolah , kebetulan di pc sekolahan itu di pasang subline text , nah pas saya pakai tampilan nya itu enak di pandang , ada fitur auto complit , contoh nya ketik html saja lalu klick tab langsung jadi struktur html secara otomatis , langsung donwload aja biar ngerasain sendiri , masih banyak lagi sih code editor di laptop ssayang nya ane gak pernah nyoba kwkw

Oke itulah cerita alat ngoding artikel kali ini , next artikel mulai belajar html Ikuti terus ya jangan sampai ketinggalan
