Mengetahui Teknologi yang di gunakan dalam website

  • CMS
  • Framework
  • CDN

Ada dua opsi yang saya berikan untuk melihat nya , yaitu lewat website , dan WebExtensions dari browser chrome . jika kamu belum tau WebExtensions browser itu adalah sistem cross-browser untuk mengembangkan browser add-ons , gak paham ya? sama hehe

Netcraft

Untuk tols pertama ini ialah website dari Netcraft , Jika kamu menggunakan netcraft kamu dapat mengetahui Os server dan Webserver

Bisa di coba sendiri https://www.netcraft.com/

Domaintools

Dengan Tools ini kamu bisa mengetahui secara terperinci sebuah domain , semisal ip domain , Negara lokasi server

Bisa coba sendiri https://www.domaintools.com/

Builtwith

Cek teknologi disini maka kamu akan mengetahui Framework , Cms , Webshosting , CDN , RSS dan lain lain secara lengkap , kamu bisa coba sendiri klick link web resmi nya https://builtwith.com/
