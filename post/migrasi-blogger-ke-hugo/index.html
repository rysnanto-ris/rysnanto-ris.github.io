Migrasi Dari Blogger ke Hugo

2021-03-20 — Penulis Rysnanto

Sabtu 2021 Resmi Migrasi Ke Ssg Hugo

Hai Apa Kabar Semua , Disini Saya mau berbagi Pengalaman Mengenai SSG Hugo ini , Sebelum nya saya menggunakan cms blogger selama 2 tahun ini , Dan Selama menggunakam blogger biasa biasa saja karena ya sudah bosen sama tampilannya , belum lagi saat mencari template blogger yang keren susah karena aku gak bisa coding jadi ngambil template orang lain heheh

Sebelum nya Apakah Kamu tau apa itu SSG? Ssg itu kepanjangan dari static site generator , jadi software penghasil halam statis , halaman statis itu hanya mempunyai tiga teknologi yaitu html , css , dan js tanpa ada backend / database

Nah Ssg yang saya gunakan ini Hugo karena rumornya ssg hugo ini paling cepat , dan ternyata benar cepat.

dibanding menggunakan blogger saat meload website terasa berat karena ada bagian server nya , sedangkan pakai ssg lebih cepat karena hanya website statis saja

Sensasi saat makai/ saat penginstallan saya radak bingung karena aku awam banget tentang git , setelah 1 minggu aku coba gagal terus dan pada hari ini saya sudah bisa . saat membuat artikel baru dan mengeditnya cukup asik karena memakai markdown bukan html

Saat ngeblog dan penginstallan hugo saya pakai terminal termux ,pasti udah tau kan termux , kamu bisa install di playstore . Bahkan kamu bisa install hugo dan ngeblog lewat terminal cukup menarik m untuk theme hugo ada banyak bisa kalian unduh di Gass Ke web Nya

Oke Itu cerita saya saat migrasi blogger ke hugo , untuk yang penasaran pengen punya blog pakai hugo stay di blog saya akan aku buatin cari penginstallan sampai deploy ke github . Bye bye
