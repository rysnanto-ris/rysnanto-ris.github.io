Code Mendasar Dalam Javascript

Untuk menampilkan Output ke dalam console javasvript menggunakan code berikut 

console.log("Rysnanto");

Menggunakan Petik dua/satu , Jika bernilai String , Sedangkan bila mana Nilai nya bernilai integer maka Jika di beri tanda petik juga tidak apa apa.

Bisa Kamu Ingat! 

  1. String Merupakan Tipe data dalam bentuk text , huruf

  2. Integer Berupa Angka

Diatas kan menghasilkan output Rysnanto ke dalam console , Nah lalu bagaimana cara kita ingin menampilkan output ke browser? , Kalian bisa memberika perintah seperti ini

document.write("Rysnanto");

Kita juga Bisa menghitung mulai dari perkalian , Pembagian , penjumlahan , pengurangan dengan memahami simbol simbol berikut ini 

1.( - ) untuk pengurangan

2.  ( +) untuk penjumlahan

3.  ( * )    di kali

4.  ( / )  pembagian

5.  (%)   Modulo/sisa bagi

Bukan hanya untuk menghitung saja , kita juga bisa menggabung String dengan simbol (+) Misalnya

console.log("Rys"+"nanto");

Apakah Kamu Mengetahui  Variabel? Variabel berisi Nilai / Data String , Integer dan lain lain . Variabel adalah sebuah nama yang mewakili sebuah nilai . Contoh code Membuat variabel javascript. pendeklarasian variabel di dalam javascript bisa menggunakan let , const , var . untuk perbedaan kedua deklrasi tadi akan saya jelaskan , saya disini menggunakan let terlebih dahulu

Lalu cara memanggil Nilai variabel Bagaimana? Mudah Saja kita cukup panggil aja nama variabel kita dan tidak usah diberi tanda kutip ya nanti malah kebaca sebagai string dan nilai variabel tidak muncul , contoh nya seperti ini 

let nama = "Rysnanto";
 console.log(nama);

Lalu apakah kita bisa mengupdate nilai variabel ? Jawaban nya bisa bila kita menggunakan deklarasi let . jika const tidak akan bisa merubah nilai variabel malah akan memuncul kan error

Lalu bagaimana cara update nilai variabel? Berikut ini contoh mengupdate nilai variabel

let nama = "Rysnanto"; 
nama = "leny"; 
console.log(nama);

Cukup memanggil nama variabel lalu isi nilai nya baru , kenapa kita tidak menuliskan deklrasi nya? tidak perlu karna kita mengupdate nya bukan membuat variabel baru
