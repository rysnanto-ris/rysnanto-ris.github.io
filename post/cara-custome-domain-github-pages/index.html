Cara Custome domain Github Pages

2021-04-03 â€” Penulis Rysnanto

Hai Gimana kabar nya , semoga sehat selalu . sesuai judul artikel disini aku mau ngasih tau cara custome domain github pages dengan domain mu sendiri . yang belum tau apa itu github pages saya jelaskan . HalamanÂ GitHubÂ (GitHub Page) sering digunakan para developer sebagai sebuah portofolio online , atau bisa juga untuk mengonline kan sebuah project halaman website namun hanya mendukung (Html,Css,Js) , secara default nya jika kamu membuat sebuah github pages kamu mendapatkan subdomain user.github.io , seperti website saya ini awalnya arysn.github.io saya custome domain menjadi Rysnanto.eu.org , semua gratis mulai dari hosting sampai domain saya gunakan gratis hahaha , karna saya kaum gratisan.

bila kamu ingin domain github pages mu menjadi domain mu sendiri , coba simak cara berikut ini . semoga berhasil kawanâœ…

  1. Login Ke github.com . jika belum mendaftar silahkan mendaftar , jika sudah masuk dasboard github pergi ke Repository github pages kamu….

  2. Klick Setting , aku disini pakai tampilan dekstop kurang lebih seperti ini

  3. Pergi ke bagian Github Pages , bisa dilihat seperti ini tampilannya , lalu masukkan domain mu di bagian yang saya kasih tanda kotak lalu klick save

  4. Sekarang kita pergi ke tempat kita membeli domain untuk konfigurasi , disini saya menggunakan hostry . setelah itu pilih domain kalian lalu pilih dns management .

Setlah itu masukkan Dns A Record Untuk Type Pilih A , subdomain (hostname) bisa kamu kosongi atau bisa isi @ , Ip addreas masukin satu satu ip dibawah ini ,untuk TTL 14440

  • 185.199.111.153

Sekarang kita tambahkan Record type CNAME , pilih type cname , untuk subdomain / hostname isi www , Domain isi dengan github pages mu misal punya saya arysn.github.io , TTL isi dengan 14440

Dan Selesai , Tunggu beberapa menit/jam untuk melihat website mu sudah terganti dengan domain mu sendiri , yang awalnya user.github.io menjadi domainmu.com
