Cara Upload Project Ke Github Lewat Termux

2021-04-06 — Penulis Rysnanto

Masih bingung cara upload/push project ke repository github , mari simak tutorial untuk cara upload termux . untuk yang harus kalian siapkan akun github , silahksn daftar / login

Setalah login dan masuk ke dasboard kalian buat sebuah repository untuk menyimpan file mu, gak tau cara untuk membuat repo baru? coba cek salah satu artikel saya mengenai custome domain github pages sudah ada penjelasan mengenai membuat repo baru .

Lanjut , Sekarang download aplikasi termux dulu tentunya , di play store ada tinggal pencet unduh . Sekarang Buka Termux kalian

Ketikan Perintah Berikut di termux

pkg install git

Fungsi dari perintah tersebut untuk menginstall git di termux kita

Jika proses telah selesai , pastikan kamu sekarang ada di folder/file yang ingin kamu upload , misalnya saya mau upload folder project yang berada di penyimpanan hp saya , cara nya ketikan seperti ini

termux-setup-storage

Fungsi dari perintah tersebut agar kita bisa membuka penyimpanan hp di termux , yuk ikuti langkah selanjutnya

Ketikan

cd storage/shared

lalu ketik ls agar muncul semua file/folder yang ada di penyimpanan kita ya!

Bisa kamu lihat pada gambar diatas , aku pengen upload file yang ada di folder project , jadi aku buka folder project dengan ketik , sesuaikan aja ya

cd namafolder

Proses Upload

lalu kita sekarang akan melakukan proses upload ke github ya , ini bagian paling penting , yang pertama yaitu ketikan

git init

fungsi nya untuk membuat repository pada file lokal yang nantinya ada folder .git

yang kedua

git add .

untuk menambahkan file/folder , bisa kamu lihat ada tanda (.) titik kan? , nah itu untuk membuat semua ikut di upload . kamu juga bisa mengupload salah satu saja / lebih spesefik contoh nya

git add index.html

Atau

git add namafolder/profil.html

Yang Ketiga

git commit -m "baru"

Fungsinya untuk membuat pesan upload yang nanti muncul di repository mu , untuk tulisan “baru” bisa kamu ganti sesuka hati

sesudah itu , langkah ke empat

git remote add origin https://github.com/namauser/namarepo.git

Itu bisa kalian sesuaikan dengan nama repo kamu ya, itu hanya contoh saja

Langkah ke lima langkah terakhir yaitu kita upload ke github kita

git push -u origin master

-u berfungsi untuk pertamakali upload file di repo , untuk sudah pernah tidak usah memakai hanya gini saja cukup

git push origin master

lalu kamu disuruh untuk mengisi username dan passowrd github kamu

dan berhasil selamat kamu sudah berhasil mengupload project mu ke repository github

Biasanya kalo ada yang eror saat upload seperti gambar di bawah ini , solusi saya yaitu dengan cara menambahkan –force seperti ini kurang lebih 

git push origin master --force
